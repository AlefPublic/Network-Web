## 📌 About The App

**N W** is a professional desktop application built with Python that combines network analysis, web scraping, security assessment, and DNS lookup into one powerful tool. It features a modern dark interface with two distinct editions - Cyber (neon green) and Dark Pro (purple glass-morphism).

The application is designed for developers, security researchers, data analysts, marketers, and anyone who needs to extract information from websites or analyze network infrastructure.

---

## ⚡ What It Does

### 🔹 Network Analysis
- Resolves IP addresses and hostnames
- Scans common ports (80, 443, 22, 21, 25, 3306, 8080)
- Retrieves WHOIS information (registrar, creation date, expiry)
- Runs ping tests to check server availability
- Performs traceroute to see network path

### 🔹 Web Data Extraction
- Extracts page title and meta description
- Grabs all header tags (H1, H2, H3)
- Lists all links found on the page
- Finds email addresses automatically
- Extracts image sources
- Displays HTML structure

### 🔹 Security Assessment
- Validates SSL certificates
- Checks certificate issuer and expiry date
- Analyzes security headers (HSTS, CSP, X-Frame-Options, X-XSS-Protection)
- Identifies missing or weak security configurations

### 🔹 DNS Analysis
- Shows A, AAAA, MX, NS, and TXT records
- Complete DNS configuration overview
- Mail server information
- Name server details

### 🔹 Reporting
- Generates comprehensive reports
- Exports to .txt format
- All data organized in one place
- Copy & paste ready
